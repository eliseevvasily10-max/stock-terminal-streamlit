# Market Terminal

Локальный `Streamlit`-терминал с данными из:

- `Yahoo Finance` через `yfinance`;
- `MOEX ISS API` для российских котировок и рыночной активности.

Что внутри:

- строка поиска по тикеру или названию компании;
- вкладка `Терминал` с карточками компаний, графиком за 5 месяцев, новостями, фундаментальными метриками и explainable `Buy / Hold / Sell`;
- вкладка `Сравнение RU vs US` с сопоставлением компаний похожих индустрий;
- вкладка `Инсайты` с narrative-выводами по выбранным акциям;
- расчеты `beta`, `CAPM`, `WACC`, `ROIC`, `Debt / EBITDA`, `Net debt / EBITDA`.

## Запуск

```bash
pip install -r requirements.txt
streamlit run app.py
```

## Примечания

- для США benchmark для `beta/CAPM` — `SPY`;
- для России benchmark для `beta/CAPM` — `IMOEX`;
- безрисковая ставка берется как прокси из `^IRX`;
- `WACC` и часть фундаментальных выводов являются приближенными оценками;
- итоговый сигнал не является инвестиционной рекомендацией.
